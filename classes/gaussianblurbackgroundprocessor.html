Class GaussianBlurBackgroundProcessor

The GaussianBlurBackgroundProcessor, when added to a VideoTrack, applies a gaussian blur filter on the background in each video frame and leaves the foreground (person(s)) untouched.

example
import { createLocalVideoTrack } from 'twilio-video';
import { GaussianBlurBackgroundProcessor } from '@twilio/video-processors';

const blurBackground = new GaussianBlurBackgroundProcessor();

createLocalVideoTrack({
  width: 640,
  height: 480
}).then(track => {
  track.addProcessor(blurBackground);
});
